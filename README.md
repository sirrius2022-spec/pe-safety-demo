## 📱 스마트폰 바탕화면 설치

**iPhone (Safari)**
1. 이 페이지를 Safari로 열기
2. 하단 공유버튼(□↑) 터치
3. "홈 화면에 추가" 선택
4. 이름 확인 후 "추가"

**Android (Chrome)**
1. 이 페이지를 Chrome으로 열기
2. 우측 상단 메뉴(⋮) 터치
3. "홈 화면에 추가" 선택
